Scheduling

In the discord there is a channel called "notice-board". This is where quests will be posted. If you are interested in a quest, react with the [1], [2], or [3] emoji. (:one:, :two:, :three:) Using only one of each. If you put a one on a quest, that is your currently requested active quest. You then check the "scheduling" channel, and see if anyone is trying to schedule a session for that quest.
